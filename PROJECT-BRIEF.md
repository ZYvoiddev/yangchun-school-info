# 阳春高新职校信息展示页项目说明

## 1. 项目目的

本项目是学生个人学习项目，主要用于练习 HTML、CSS、JavaScript 静态网页开发，并尝试探索中职学校官网的信息架构、招生内容组织和移动端展示方式。

项目当前定位为“公开信息展示页”，不是学校官方网站，也不是正式招生报名系统。

## 2. 项目边界

- 本项目未获得学校官方授权。
- 本项目不代表学校发布正式招生信息。
- 本项目不提供真实报名功能。
- 本项目不收集学生或家长的真实个人信息。
- 本项目内容基于公开资料整理，具体招生政策、收费标准、报名方式等以学校官方发布为准。

## 3. 已完成内容

当前网站已完成 7 个页面：

- 首页
- 学校概况
- 专业介绍
- 校园环境
- 招生就业
- 联系我们
- 资料来源与免责声明

已实现功能：

- 移动端适配
- 手机端折叠菜单
- 专业分类筛选
- 演示咨询表单
- 非官方声明
- 资料来源与使用边界说明

## 4. 已规避的风险

- 不使用“官方网站”“官方报名入口”等容易误导的表述。
- 不写未经确认的学费、招生计划、报名方式、资助政策。
- 不承诺“保证就业”“保证升学”“包就业”等结果。
- 不展示教师私人联系方式。
- 不展示学生真实照片或个人信息。
- 不直接搬运新闻图片原图，第一期使用本地示意图。
- 表单只做前端演示，不提交到学校，也不代表报名成功。

## 5. 当前技术方案

- 技术栈：HTML + CSS + JavaScript
- 部署方式：GitHub Pages
- 仓库地址：https://github.com/ZYvoiddev/yangchun-school-info
- 访问地址：https://zyvoiddev.github.io/yangchun-school-info/

## 6. 如果学校认可，后续可升级

如果学校老师或相关部门认为这个项目有参考价值，后续可以在获得授权后升级为更正式的版本：

- 替换为学校官方 Logo、校徽和校园实景照片。
- 补充学校正式招生简章、专业代码、学费标准和资助政策。
- 增加新闻公告、校园动态、校企合作、学生作品等栏目。
- 增加后台或内容管理方式，方便老师维护。
- 改用学校正式域名或服务器部署。
- 根据学校宣传口径统一文案和视觉风格。

## 7. 建议查看方式

建议先用手机打开网站首页，重点查看：

- 顶部非官方声明是否清楚
- 首页是否能快速理解学校基本信息
- 专业介绍是否便于初三学生和家长阅读
- 招生就业页是否避免了误导性承诺
- 资料来源与免责声明页是否说明了项目边界
