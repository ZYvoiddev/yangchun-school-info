# 阳春高新职校信息展示页

这是一个学生个人学习项目，用于练习中职学校招生信息展示页的 HTML/CSS/JS 实现。

## 项目定位

- 非学校官方网站
- 未获得学校官方授权
- 内容基于公开资料整理
- 招生政策、收费标准、报名方式等以学校官方发布为准

## 页面

- `index.html` 首页
- `about.html` 学校概况
- `majors.html` 专业介绍
- `campus.html` 校园环境
- `admission.html` 招生就业
- `contact.html` 联系我们

## 技术

- HTML
- CSS
- JavaScript
- GitHub Pages 静态部署

## 本地预览

直接用浏览器打开 `index.html` 即可。
